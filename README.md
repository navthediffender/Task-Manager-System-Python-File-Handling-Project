# Task Manager System – Python File Handling Project

## Project Overview

Developed a multi-user task management application in Python with login, user registration, task assignment, task editing, task deletion, completion tracking, reporting, and file handling.

This project was completed as part of the HyperionDev / The University of Manchester Cyber Security Bootcamp. It helped strengthen my practical Python programming skills, especially working with text files, user-based workflows, data validation, and structured problem-solving.

## Features

- User login system using `user.txt`
- Admin-only user registration
- Add new tasks and assign them to users
- View all tasks
- View tasks assigned to the logged-in user
- Mark tasks as complete
- Edit task username or due date
- Delete selected tasks
- Generate task reports
- Generate user overview reports
- Display statistics for admin users

## Files Included

- `task_manager2.py` – main Python program
- `tasks.txt` – stores task records
- `user.txt` – stores usernames and passwords
- `task_overview.txt` – generated task statistics
- `user_overview.txt` – generated user statistics
